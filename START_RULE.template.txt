# START RULE (Room Panel Offline Demo)

- Критичный стиль: кратко, по делу, с ожидаемым результатом.
- План → подтверждение: перед крупной работой — короткий план, после — сводка изменений.
- Не дублировать код/доки в сообщениях: ссылаться на файлы и строки.
- Использовать «Цепочку входа»: `SYSTEM_OVERVIEW.md` → `QUICK_RETURN.md`.
- По умолчанию офлайн: любые сетевые действия только по явному согласию.
- Качество UI для презентаций: крупная типографика, стекло, анимации.
